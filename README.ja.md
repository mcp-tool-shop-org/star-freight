# スターフレイト

あなたはかつて軍のパイロットでした。しかし、その後は恥をかきました。そして今、あなたは劣悪な船の船長であり、何の信用も得られず、さらに、あなたが赴任する前からすでに動き出している星系を任されています。

**Star Freight**は、政治的に分裂した星系を舞台に、物資を輸送する戦略シミュレーションRPGです。五つの文明、一つの経済、そして二度と同じ人生を送らせてくれない四つの真実。

---

## そのゲーム

あなたは、文化が交渉のやり方を変えるような港に停泊します。地形が戦闘方法を変えるような航路を選びます。乗組員を雇うことで、できること、アクセスできること、そして負うべき義務が変わります。一見単純に見える契約を結びますが、書類手続きが進むと、物資の不足によって価格が変動したり、積載物が証拠品であることが判明したりすることがあります。

あなたは選ばれた者ではありません。あなたは、組織が貨物を押収し、物資不足が航路を変化させ、乗組員が様々な義務を抱え、そして危険な真実が日常的な業務を通して明らかになる中で、船を動かし続けるために奮闘する船長なのです。

毎回、プレイするごとに異なる「船長」としての人生が体験できます。それは、選択したクラスだけでなく、乗組員、航路、リスク、そしてあなたの立ち位置が、あなた自身を形作っていくからです。

## なぜそれが違って感じられるのか

多くのRPGは、システムを横に並べて配置しています。一方、Star Freightは、それらのシステムを連携させ、相互に作用させることを可能にします。

**クルーは絶対的なルールです。** サルは単なる「修理速度+10%」のボーナスではありません。サルこそが、なぜケス駅に停泊できるのか、なぜあなたの船が緊急修理能力を持っているのか、そしてなぜ医療物資が季節と一致しないことに気づいたのか、その理由なのです。サルを失うと、同時に3つのシステムが機能しなくなります。

**戦闘はキャンペーンイベントです。** 戦闘に突入して、また戦闘から離脱するという形になります。勝利すると、サルベージクレジットと派閥からの評価が得られます。撤退には、廃棄された貨物と、「逃げ出す」人物としての評判の低下という代償があります。敗北すると、貨物が奪われ、乗組員が負傷し、船は修理が必要な状態で最寄りのステーションまでゆっくりと移動することになります。どのような結果になっても、次の取引の判断に影響を与えます。

**文化とは、社会的な論理である。** ケス族は単に価格が異なるだけでなく、季節ごとの生物学的なサイクルがあり、それが贈り物をする意味、取引を強要することが侮辱と見なされるタイミング、そして外部からの訪問者がどの範囲に留まるべきかを決定します。ヴェシャン族は単に争うだけではありません。彼らは挑戦し、それを拒否することは敗北よりも悪いとされています。知識とは、単なる書物ではありません。それは、状況を読み解く能力です。

**調査は、日々の活動から生まれるものです。** 医療物資が季節と合っていないことに気づくのは、あなたがそれを輸送しているから。積荷リストの不一致を見つけるのは、あなたが難破船から回収したから。あなたのクルーメンバーであるケスは、そのパターンを理解できるのは、彼女が同じサプライヤーに対する借金を覚えているから。陰謀は、自ら姿を現すことはありません。あなたは、仕事をしているうちに、偶然それに巻き込まれるのです。

## 世界

5つの文明が、「閾（きょう）」と呼ばれる恒星系を共有しています。

**テラン同盟** - 官僚的な人間による政府組織。彼らはあなたを恥ずかしい目に合わせた。再び活動を始めるには、許可、忍耐、そしてプライドを捨てることが必要です。安全な市場は存在するものの、利益率は低く、書類手続きも煩雑です。

**ケス同盟 (The Keth Communion)**：生物学的なサイクルに基づいて活動する、節足動物の集団。辛抱強く、観察力があり、しかし侮られた場合は容赦しない。季節の変化を理解していれば、この組織との関係は非常に有利になる可能性があります。しかし、理解が不足していると、評判を大きく損なうことにもなりかねません。

**ヴェシャン諸侯国** - 爬虫類のような種族が築いた封建的な国家群。彼らは名誉と義務を重んじ、形式的な戦いを好み、直接的な交易を行い、そして何事も忘れません。 「負債記録」は単なる飾りではありません。それは彼らの交渉における重要な要素なのです。

**オーリン・ドリフト** - 移動式の仲介業者集団。政策上は中立を保ち、しかしその仕組み自体は利益を追求するよう設計されています。彼らは誰とでも取引を行い、あらゆる情報を把握しており、その両方に対して料金を請求します。彼らを利用しないことでコストを削減できますが、彼らとの良好な関係を失うことは、より大きな損失につながる可能性があります。

**The Sable Reach** — 海賊勢力、古代技術の回収業者、そしてCompactが忘れ去りたい人々。 法も慣習もない。 返金はなし。 システム内で最もリスクが高く、同時に最も高い報酬が得られる場所。

## 船長のタイプは3つ

同じ世界。 しかし、3つの異なるプレッシャー。

**規律ある船長 (Relief captain)。** 護送隊の規律、信頼に基づくアクセス、公的な制裁。 人々を養い、正当性の罠にかかる。 遅延と能力不足を恐れる。

**グレーゾーンの船長 (Gray runner)。** 形式的な権限の利用、タイミングの悪用、組織的なリスク。 読みにくさを利用して利益を得る。 没収と暴露を恐れる。

**名誉を重んじる船長 (Honor captain)。** 直接対決、派閥内の政治、評判の変動。 問題は直接解決する。 エスカレーションと脆弱な支援ネットワークを恐れる。

これらはクラスではなく、あなたの選択があなたを形作った結果です。

## クイックスタート

```bash
# Clone and install
git clone https://github.com/mcp-tool-shop-org/star-freight.git
cd star-freight
pip install -e ".[tui]"

# Start a new game
starfreight new "Captain Nyx" --type merchant
starfreight tui
```

**操作:** `D` ダッシュボード | `C` 乗組員 | `R` 航路 | `M` マーケット | `T` 基地 | `J` ジャーナル | `F` 派閥 | `B` 購入 | `S` 販売 | `G` 移動 | `A` 進む

## 現在の状態

Star Freightは、設計コンセプトではなく、実績のある製品です。

| | 数 |
|---|---|
| ステーション | 9 |
| 宇宙航路 | 14 |
| 交易品 | 18 |
| 乗組員 | 5人 + プレイヤー |
| 契約 | 9 |
| 遭遇イベント | 6 |
| 調査の糸口 | 4 |
| 成功判定 | 2,200+ |

ゲームの主要部分（vertical slice）は、以下の3つの検証基準をすべてクリアしました。 Golden Path（継続的な船長としての人生）、Encounter（異なるキャンペーン状態を持つ3つのブランチ）、Economy（崩壊することなくプレッシャーが持続する経済システム）。

以下の3つの拡張パックがリリースされています。 Working Lives（人間の描写）、Houses, Audits, and Seizures（組織的なプレッシャー）、Shortages, Sanctions, and Convoys（資源の管理）。

船長のプレイスタイルによる分岐が確認されました。 3つのプレイスタイルは、異なるルート、異なる交易の組み合わせ、異なる戦闘スタイル、異なる失敗のパターン、そして異なる船長の個性をもたらします。

## 技術

Python 3.11以上。高機能なテキストUI。乗組員の管理、グリッド形式の戦闘、文化知識、調査、キャンペーンの連携。2200件以上のテスト。

外部のAI依存はありません。 クラウドサービスも使用しません。 あなたのコンピュータ上で動作します。

**セキュリティモデル:** Star Freightは、シングルプレイヤーのオフラインゲームです。ローカルのセーブファイルのみを使用します。ネットワークにアクセスしたり、テレメトリを収集したり、認証情報を保存したり、ユーザーアカウントを必要としたりすることはありません。依存関係は、Typer、Rich、Textualであり、これらはすべてメンテナンスが良好で、ネイティブコードは使用していません。詳細については、[SECURITY.md](SECURITY.md) を参照してください。

## 基本的なルール

次に何をビルドすべきか迷った場合：

- それは、4つの基本的な真実のいずれかを強化しますか？
- それは、船長としての人生を豊かにしますか？
- それは、プレイヤーが感情的に反応できるような選択肢を生み出しますか？

もしそうでない場合は、後回しにできます。
