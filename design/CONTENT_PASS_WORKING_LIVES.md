# CONTENT PASS — Working Lives Pack (Phase 7A)

> Every file must answer three questions:
> 1. What decision does this create?
> 2. What truth does this reinforce?
> 3. Why isn't an existing content unit already doing this?

## Goal
- What we are adding: Civilian life, ritual commerce, non-pirate conflict, institutional people, goods with political meaning, investigation born from daily work
- What truths this batch must reinforce: All four existing truths. No new systems.

## Included units
- Stations: Mourning Quay (Keth hospice-port)
- Lanes: Pilgrim's Ribbon (ceremonial safe lane), Cinder Span (dangerous salvage lane)
- Crew: Sera Vale (Compact quartermaster, document expert)
- Contracts: Witness Run (observer transport), Cold Lantern Freight (sacred cargo deadline)
- Encounters: Hearth Right Challenge (Veshan lane dispute)
- Threads: Ghost Tonnage (missing aid shipments)
- Goods: Brood Silk (Keth ceremonial textile), Black Seal Resin (Veshan oath material)

## Proof coverage
- Crew truth: Sera changes trade finesse, document reading, and investigation. Thal/Varek interactions with new content shift existing postures.
- Combat truth: Hearth Right is social-conflict-to-combat. Cinder Span forces tactical terrain decisions.
- Culture truth: Mourning Quay, Pilgrim's Ribbon, Cold Lantern, and both goods carry cultural weight. Brood Silk mishandling is an insult. Black Seal Resin ties economy to house honor.
- Plot truth: Ghost Tonnage emerges from trade patterns. Witness Run can branch into investigation. Sera opens document-based angles.
- Economy truth: Two new goods with distinct spread/risk profiles. Two new contracts with different risk shapes. Two new lanes with different cost/reward profiles.

## Risks
- Drift risk: Mourning Quay could become lore-heavy. Keep it mechanical — what decisions does docking here create?
- Repetition risk: Pilgrim's Ribbon could feel like a slower version of existing safe lanes. Its scrutiny pressure must be distinct.
- Overlap: Ghost Tonnage must not duplicate Medical Shipment. Different scope (logistics fraud vs tech smuggling), different sources.
- What to cut first if weak: Pilgrim's Ribbon (least mechanically distinct). Black Seal Resin (most niche good).

## Exit criteria
- Content complete: all units implemented in code
- Validator passes: graph connected, all goods referenced, no orphans
- Distinctiveness check: each unit creates behavior the slice didn't have
- Golden Path replay: a session touching new content still flows as one captain life
