# CONTENT PASS — Houses, Audits, and Seizures (Phase 7B)

> Every file must answer three questions:
> 1. What decision does this create?
> 2. What truth does this reinforce?
> 3. Why isn't an existing content unit already doing this?

## Goal
- What we are adding: Institutional pressure — the feeling that power reaches the captain through paperwork, claim, legitimacy, and delayed force
- What truths this batch must reinforce: All four existing truths, expressed through governance, not violence

## Thesis
Power has memory here. The captain can be ruined, delayed, claimed, or repurposed by systems that never needed to hate them personally.

## Included units
- Stations: Registry Spindle (Compact jurisdiction hub)
- Lanes: White Corridor (monitored priority route), Grain Eclipse (semi-legal bypass)
- Crew: Nera Quill (former house registrar, institutional violence expert)
- Contracts: Bonded Relief Run (administrative fragility), Claim Courier (legal race)
- Encounters: Seizure Notice (paperwork as campaign event)
- Threads: Paper Fleet (institutional disappearance)
- Goods: Bond Plate (legitimacy token), Reserve Grain (politically governed cargo)

## Design law
Do NOT turn this into bureaucracy theater. The captain should not spend the game filling forms. The point is that paperwork changes routes, legitimacy changes encounters, documentation changes leverage, and delay changes survival. The pressure must stay dramatic, not clerical.

## Proof coverage
- Crew truth: Nera changes how the captain relates to official truth. Sera + Nera together create document intelligence depth.
- Combat truth: Seizure Notice can escalate to combat. White Corridor makes combat more consequential.
- Culture truth: Registry Spindle makes Compact culture feel like law, not just bureaucracy. Bond Plate makes legitimacy portable.
- Plot truth: Paper Fleet deepens Ghost Tonnage. Claim Courier can branch into investigation.
- Economy truth: Reserve Grain is politically governed. Bond Plate changes cargo risk. Bonded Relief Run creates administrative fragility.

## Risks
- Drift risk: Bureaucracy theater. Every administrative element must create a player decision, not a menu.
- Repetition risk: Nera could feel too similar to Sera. Must be distinct: Sera = cargo law. Nera = institutional power.
- Overlap: Paper Fleet must not duplicate Ghost Tonnage. Different scope: GT is logistics fraud, PF is institutional disappearance.
- What to cut first if weak: Grain Eclipse (least distinct lane). Bond Plate (could be too abstract).

## Exit criteria
- Content complete: all units implemented in code
- Validator passes: graph connected, goods referenced, no orphans
- Distinctiveness check: every unit creates behavior the previous packs didn't have
- Pressure test: at least one session where institutional threat changes route/contract choice without combat
