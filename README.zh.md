# Star Freight

你曾是一名军用飞行员。然后，你成为了一个耻辱。现在，你是一名船长，拥有一艘状况不佳的飞船，没有任何影响力，并且所处的星系在你的到来之前就已经在运转。

**Star Freight** 是一款以文字为基础的战术商贸角色扮演游戏，讲述了在政治分裂的星系中运输货物的故事。 包含五个文明，一个经济体系，以及四个真理，这些真理将确保你无法重复第二次人生。

---

## 游戏

你会在不同的港口停靠，每个港口的文化都会影响你的谈判方式。 你会选择不同的航线，不同的地形会影响你的战斗方式。 你会雇佣船员，他们会改变你能够做什么，你能够访问什么，以及你欠什么。 你会接受看似简单的合同，但当文件出现问题，当供应短缺导致价格上涨，或者货物被发现是证据时，情况就会变得复杂。

你不是被选中的人。 你是一名努力维持飞船运转的船长，而机构正在没收货物，短缺正在重塑航线，船员带来了各种义务，而危险的真相则在日常工作中浮出水面。

每一次航行都将塑造你成为一个不同的船长。 这不是因为你选择了某个职业，而是因为你的船员、航线、风险和声誉塑造了你。

## 它与众不同的原因

大多数角色扮演游戏将各种系统并列放置。 Star Freight 让这些系统相互作用。

**船员是约束力。** Thal 不仅仅是一个+10%的维修加成。 Thal 是*你*能够停靠在 Keth 港口的原因，*你*的飞船拥有紧急维修能力的原因，以及*你*注意到医疗货物与季节不符的原因。 失去 Thal，三个系统就会同时失去功能。

**战斗是一种战役事件。** 你不会进入一场战斗然后立即退出。 胜利会带来回收的信用和派系声望。 撤退会消耗掉的货物和“逃兵”的名声。 失败意味着被没收的货物、受伤的船员，以及一艘勉强漂浮到最近港口的飞船，需要支付高额费用。 每一个结果都会改变你下一次的贸易决策。

**文化是社会逻辑。** Keth 人不仅仅是价格不同。 他们拥有一个季节性的生物日历，它会改变赠送礼物的含义，什么时候提出交易是一种冒犯，以及什么时候外来者会被限制在外部区域。 Veshan 人不仅仅是战斗。 他们会挑战，而拒绝比失败更糟糕。 知识不是一个代码，而是观察周围的环境。

**调查从生活中产生。** 你会注意到医疗货物与季节不符，是因为你一直在运输它。 你会发现清单上的差异，是因为你打捞了一艘残骸。 你的 Keth 船员会解读这种模式，因为她记得她欠同一家供应商的债务。 阴谋不会主动暴露。 你会通过完成工作而偶然发现它。

## 世界

五个文明共享一个名为“阈”的星系。

**地球联邦 (The Terran Compact)**：官僚主义的人类政府。 他们让你蒙上耻辱。 重新获得他们的信任需要许可证、耐心和放下自尊。 市场安全，利润微薄，文件繁琐。

**Keth 共同体 (The Keth Communion)**：一个遵循生物日历的节肢动物集体。 谨慎、观察，并且一旦受到冒犯，就会造成毁灭性打击。 如果你了解季节，这里可以获得最高的利润。 如果你不了解，你的声誉会迅速崩溃。

**Veshan 王国 (The Veshan Principalities)**：痴迷于荣誉和债务的爬行动物封建家族。 他们会正式战斗，直接交易，并且记住一切。 债务账簿不仅仅是装饰，它是筹码。

**Orryn 流浪者 (The Orryn Drift)**：一个移动的经纪人文明。 政策上是中立的，但设计上是盈利的。 他们与所有人交易，了解一切，并对所有服务都收费。 放弃他们可以节省资金。 失去他们的信任会付出更大的代价。

**黑沙边境**——海盗势力、古代科技回收者，以及那些“联邦”宁愿忘记的人。这里没有法律，没有习俗，也无法退款。这是整个系统中风险最高、回报也最高的地区。

## 三种类型的船长

同一个世界，三种不同的压力。

**保障型船长。** 强调运输安全，基于信任的准入机制，公开的后果。你负责让人们吃饱，但也会因为遵守规则而陷入困境。你害怕延误和运力不足。

**灰色走私者。** 利用规章漏洞，利用时机，承担机构风险。你通过保持神秘来赚钱。你害怕被没收和暴露。

**荣誉型船长。** 直接对抗，涉及家族政治，声誉波动。你通过直接对话解决问题。你害怕事态升级和缺乏支持。

这些不是职业，而是你的选择将你塑造成了什么。

## 当前状态

“星际货运”是一款经过验证的产品，而不是一个设计概念。

|---|---|

| 贸易商品 | 18 |
| 船员 | 5 + 玩家 |
| 合同 | 9 |
| 遭遇 | 6 |
| 调查线索 | 4 |
| 测试通过 | 2,161 |

核心功能已经通过所有三个验证标准：黄金路径（持续的船长生涯）、遭遇（三个分支，不同的游戏进程）、经济（压力可持续，不会导致无聊）。

已发布三个扩展包：工作生活（展现人物的真实面貌）、家族、审计和没收（机构压力）、短缺、制裁和运输队（有管理的稀缺）。

船长发展路线的多样性已经得到验证：三种不同的立场会产生不同的路线、不同的贸易组合、不同的战斗风格、不同的失败模式，以及不同的船长身份。

## 技术

Python 3.11+。 丰富的文本用户界面。 衍生自 [Portlight](https://github.com/mcp-tool-shop-org/portlight)——一个拥有 1832 个测试的航运策略引擎，提供贸易模拟、经济模型和世界状态架构。 “星际货运”在此基础上增加了船员管理、网格战斗、文化知识、调查和战役编排等功能。

不依赖任何外部人工智能。 不使用云服务。 运行在您的计算机上。

## 基本原则

当不确定下一步该做什么时：

- 它是否强化了四大基本原则之一？
- 它是否丰富了船长的生活？
- 它是否创造了一个玩家可以感知的选择？

如果不是，可以暂时搁置。
